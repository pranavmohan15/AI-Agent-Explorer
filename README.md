# 🤖 AI Agent Explorer

A simple Streamlit web app demonstrating multiple AI agents using the OpenAI API.

The app currently includes:

- 🍳 Recipe Generator
- 🎬 Movie Information Generator

Users can enter prompts and receive structured AI-generated responses.

## Tech Stack

- Python
- Streamlit
- OpenAI API
- Pydantic

## Features

- AI-powered recipe generation
- AI movie information lookup
- Structured JSON outputs validated with Pydantic
- Interactive UI with Streamlit

## Setup

Install dependencies:

```
pip install -r requirements.txt
```

Run the app:

```
streamlit run app.py
```

Enter your OpenAI API key in the sidebar and start exploring the AI agents.
